# docx

First-party Grok Build skill for creating and editing Word documents (`.docx` / `.dotx`).

## Install

```text
/plugin install docx@xai-official
```

Or accept the inline install CTA when your prompt mentions Word / `.docx` documents.

## Usage

The skill triggers on Word document work (create, edit, inspect, comments, redlines). Scripts live under `skills/docx/scripts/`.
