# pptx

First-party Grok Build skill for creating and editing PowerPoint presentations (`.pptx`).

## Install

```text
/plugin install pptx@xai-official
```

Or accept the inline install CTA when your prompt mentions PowerPoint / `.pptx` / pitch decks.

## Usage

The skill triggers on presentation work (create, edit, inspect `.pptx`). It ships scripts under `skills/pptx/scripts/` and templates for common deck layouts.
